Dodge It!!
Dodge It!!, Unity oyun motoru kullanılarak geliştirilmiş, minimalist ve yüksek tempolu bir 2D "top-down" mobil arcade oyunudur.

🚀 Oyun Hakkında
Bu proje, oyuncuların reflekslerini zorlayan, basit ama bağımlılık yapıcı mekaniklere sahip bir mobil deneyim sunmak amacıyla geliştirilmiştir. Oyunun temel amacı, giderek zorlaşan engellerden kaçarak en yüksek skora ulaşmaktır.

🛠️ Teknolojiler ve Araçlar
Proje geliştirme sürecinde aşağıdaki teknoloji ve tasarım araçları kullanılmıştır:

Oyun Motoru: Unity (6.3 LTS)

Programlama Dili: C#

Grafik Tasarım: Photoshop, Illustrator

Ses Efektleri: Reaper

Versiyon Kontrol: Git & GitHub

🎮 Öne Çıkan Özellikler
Minimalist Tasarım: Göz yormayan, yüksek kontrastlı ve erişilebilir kullanıcı arayüzü (UI).

Akıcı Mekanikler: Mobil cihazlar için optimize edilmiş, sezgisel kontrol şeması.

Dinamik Zorluk: Oyuncunun süresine ve performansına göre şekillenen zorluk seviyesi.
